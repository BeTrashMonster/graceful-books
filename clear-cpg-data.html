Instructions:

  1. Make sure your Graceful Books app is running (npm run dev)
  2. Make sure you're logged in to the app
  3. Open your browser's DevTools (F12)
  4. Go to the Console tab
  5. Copy and paste the commands below

Browser Console Commands

1. Get Your Company ID

const companyId = window.getCompanyId();
console.log('Company ID:', companyId);

2. Check Current Data (Optional)

window.getCPGDataCounts(companyId).then(counts => {
  const total = Object.values(counts).reduce((sum, count) => sum + count, 0);
  console.log('Total CPG records:', total);
  console.log('Details:', counts);
});

3. Clear All CPG Data

window.clearCPGData(companyId).then(result => {
  const total = Object.values(result).reduce((sum, count) => sum + count, 0);
  console.log('‚úÖ Cleared', total, 'records');
  console.log('Details:', result);
});

One-Liner (All Steps Combined)

window.clearCPGData(window.getCompanyId()).then(r => console.log('Cleared:', r));
